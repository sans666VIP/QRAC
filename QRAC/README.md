# QRAC - 文件图像编码工具

一个强大的工具，可以将文件编码为图像，也可以从图像中解码回原文件。

## 功能特点

- 📁 **文件转图像**：把任何文件变成图片
- 🔍 **图像转文件**：从图片中提取原始文件
- 🛡️ **纠错功能**：内置错误纠正，数据更安全
- 🖼️ **多格式支持**：支持PNG、BMP等格式

## 快速开始

### 系统要求
- Windows 系统
- Visual Studio 2019 或更新版本

### 编译方法
1. 用 Visual Studio 打开 `QRAC.vcxproj`
2. 按 F7 编译
3. 运行生成的 .exe 文件

### 使用方法
程序启动后有菜单：
1. 编码：把文件变成图片
2. 解码：从图片还原文件  
3. 校正：修复损坏的图片
